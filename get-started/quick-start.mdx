## Lets build your agent right now.

Building voice AI agents has never been easier. Get started in just a few simple steps to build the AI that can automate anything, repeatibly, every single time.

<Steps>
  <Step title="Pick a voice" icon="waveform" stepNumber={1}>
    Choose from thousands of pre-built voices, import from leading providers, or clone a custom voice in minutes to match your brand.

    <Frame caption="Voice selection interface">
      ![Quick Start Voice Example](/assets/quick-start-voice.gif)
    </Frame>
  </Step>
  <Step title="Add your knowledge" icon="book" stepNumber={2}>
    Upload FAQs, SOPs, and docs to the knowledge base.\
    Phonely uses retrieval (RAG) to answer policy and product questions anywhere in the call.

    <Frame caption="Knowledge base upload view">
      ![Quick Start Knowledge Base Example](/assets/quick-start-knowledgebase.gif)
    </Frame>
  </Step>
  <Step title="Design the flow" icon="workflow" stepNumber={3}>
    Use the visual Flow Builder to map greetings, questions, transfers, API calls, scheduling, and post-call actions. Learn best pracrices of flow building here.

    <Frame caption="Flow builder canvas">
      ![Phonely flow builder canvas](/assets/flow-builder.png)
    </Frame>
  </Step>
  <Step title="Capture variables" icon="list" stepNumber={4}>
    Define what to collect (e.g., name, DOB). Mark key fields **required** and **confirm** spelling.\
    Use auto-gather to collect data exactly when the block needs it.

    <Frame caption="Variable configuration panel">
      ![Variable configuration settings in Phonely](/assets/variable-settings.png)
    </Frame>
  </Step>
  <Step title="Transfer & escalate" icon="phone-forwarded" stepNumber={5}>
    Add **warm transfers** (with voicemail detection and fallback) or **cold transfers**.\
    Route by intent, person, or department to keep every call moving toward resolution.

    <Frame caption="Transfer configuration options">
      ![Warm and cold transfer configuration](/assets/transfer-options.png)
    </Frame>
  </Step>
  <Step title="Integrate actions" icon="plug" stepNumber={6}>
    Use pre-built integrations like **Google Calendar**, or call your APIs with the **API Request** block.\
    Pass variables such as `@name` and `@dob` dynamically.

    <Frame caption="API Request block configuration">
      ![API request configuration in Phonely](/assets/api-request.png)
    </Frame>
  </Step>
  <Step title="Publish & test" icon="rocket" stepNumber={7}>
    Click **Publish**, hit **Test**, or call your provisioned number to hear your agent live.\
    Iterate fast and deploy instantly.

    <Frame caption="Testing an agent in the dashboard">
      ![Quick Start Dashboard Example](/assets/quick-start-agent-dashboard.gif)
    </Frame>
  </Step>
</Steps>

Once you've built and tested your first agent, you'll start to see how flexible Phonely really is.
Every block, variable, and integration can be iterated on in seconds, so you can continuously improve performance and add new capabilities as you scale.

From here, you're ready to explore the deeper concepts that power every Phonely agent: flows, outbound calling, and A/B testing.

## FAQ

<AccordionGroup>

  <Accordion title="How long does it take to build my first agent?">
    Most users build a working voice agent in under **10 minutes** using the Quick Start tutorial.  
    If you’re connecting integrations or APIs, expect another few minutes to authenticate and map variables.
  </Accordion>

  <Accordion title="What if my business uses multiple software tools?">
    You can connect multiple integrations inside a single flow.
    For example, pull appointment data from Google Calendar, send confirmations via SMS, and log the interaction in HubSpot, all in one call.
    <ProTip>
      If you don’t see your integration listed, you can **import any API directly from a cURL command** to make it work instantly.
    </ProTip>
  </Accordion>

  <Accordion title="Can I test my agent before deploying it?">
    Yes. Use the **Test** button on the agent design page or call your assigned Phonely number.
    You can hear the agent in real time, confirm it follows your call flows, and review analytics immediately after.
  </Accordion>

  <Accordion title="What if the AI gets something wrong on a call?">
    Check the **Call History** tab to view transcripts, summaries, and reasoning for every decision the AI made.  
    From there, you can retrain your flow or tweak prompts.  
    Each call is logged with topic, sentiment, and variable data for easy troubleshooting.
  </Accordion>

  <Accordion title="How do I connect my existing phone number?">
    You can forward, port, or provision a new number with Phonely.  
    Most users forward their existing line to their Phonely number during setup to start testing immediately.
  </Accordion>


</AccordionGroup>
